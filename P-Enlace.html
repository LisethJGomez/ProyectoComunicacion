Presupuesto de Enlace

El presupuesto de enlace es el cálculo que nos permite saber si una señal llegará con suficiente potencia desde una antena transmisora hasta una receptora. Se calcula así:

Potencia recibida = Potencia transmitida + Ganancias - Pérdidas

Este valor se expresa en decibelios-milivatios (dBm).
